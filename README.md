# My Personal Portfolio - A React-Based Website

Welcome to my personal portfolio project! This is a website that showcases my skills and projects as an entry-level front-end developer. The website is built using React. The project was done while studying with Code First Girls.

# Features

A clean and modern design that showcases my work in an elegant and professional manner
A responsive layout that adjusts to different screen sizes, making it accessible on all devices
Sections that highlight my skills, education, experience, and projects
Interactive elements, including a contact form and links to my social media profiles

# Technologies Used

React
HTML and CSS
JavaScript
emailjs
wheatear API

# How to Use

To view my personal portfolio, simply navigate to https://rockenot22.github.io/portfolio-react/
